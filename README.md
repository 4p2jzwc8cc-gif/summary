# Random Task Generator

**Автор:** [Ваше Имя Фамилия]
**Язык:** Python 3.x

## Описание
Простое графическое приложение для генерации случайных задач из предопределённого или пользовательского списка. Позволяет сохранять историю, добавлять новые задачи, фильтровать по типу.

## Возможности
- Генерация случайной задачи.
- Фильтрация по типу (учёба, работа, спорт, отдых).
- Добавление пользовательских задач с валидацией.
- Сохранение истории в JSON.
- Загрузка истории при запуске.

## Как запустить
1. Установите Python 3.
2. Клонируйте репозиторий.
3. Запустите:
   python main.py

## Примеры использования
- Нажмите «Сгенерировать задачу» — появится случайная задача.
- Добавьте новую задачу через форму и выберите тип.
- Используйте фильтр для просмотра задач по категориям.
